## Getting Started

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd ctf-blog
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
